HOW THIS WORKS
This tool uses yt-dlp under the hood — a powerful open-source downloader.
Since browsers can't run yt-dlp directly, clicking Generate Command builds a ready-to-run
terminal command. Paste it in your terminal to download the selected format locally.

Install yt-dlp: pip install yt-dlp  |  or: brew install yt-dlp
